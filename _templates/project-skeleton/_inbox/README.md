# _inbox — Unsorted, awaiting triage

Content that was ingested but doesn't yet have a clear home lands here. It is the
catch-all that guarantees **nothing falls through** during ingestion.

## Rules

- Every item here is temporary. The goal is for this folder to be **empty**.
- Ingestion is not "done" until every item has been triaged into a real slot
  (`CLAUDE.md`, `context/`, `planning/`) or explicitly pruned (recorded in the ledger).
- When you triage an item, update `_archive/INGEST-LEDGER.md` so its verdict changes
  from `inbox` to its final destination.

This file is a placeholder so the folder exists in git. Delete it once the project
is established and the inbox is empty.
